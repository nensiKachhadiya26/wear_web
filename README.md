# WearWeb Frontend

This is the frontend of WearWeb project built using React and Vite.

## Features
- User Interface
- API Integration
- Responsive Design

## Technologies Used
- React.js
- Vite
- CSS

## Installation

1. Clone repository

git clone https://github.com/yourusername/wear_web.git

2. Install dependencies

npm install

3. Run project

npm run dev
